14

a (= anchor = 닻 = 링크) : 클릭 시 링크(해당 페이지) 열림

href (= hypertext reference = 태그 참조) : 링크(해당 페이지) 설명 시 사용

target="_blank" : 새 창에 페이지 열림

title : 어떤 내용을 담고 있는지 보여줌

-> a href="링크" target="_blank" title="링크 제목" 링크 제목 /a

참조: HTML links

13

title : 웹페이지 제목지정

-> title 제목 /title

참조: HTML title

meta charset="utf-8" : 영어 이외 다른 언어 사용가능

head : 웹 페이지를 설명하는 태그

-> head title 제목 /title meta charset="utf-8" /head

body : 웹 페이지 본문 태그

! doctype html, html = head 와 body 를 감싸는 태그 : HTML로 만들어졌음을 표현

-> ! doctype html html /html

12

li (= list = 목록) : 목차 만들어줌

-> li 목차 1 /li li 목차 2 /li

ul (= unordered llist) : 목록과 목록 간의 구분을 해줌

-> ul li 목차 1-1 /li /ul ul li 목차 2-1 /li /ul

ol (= oreded line) : 목록과 목록 간의 구분을 해줌 + 순서 추가

11


img (= image = 이미지) : 그림 삽입

src (= source = 속성) : 그림 출처 불러오기

-> img src ="그림출처" 또는 img src ="현재 파일과 같은 폴더에 있는 이미지명"

* public domain image 에서 이미지 따오기

width : 이미지 크기 조정 (숫자 or %)

-> img src="그림출처" width="숫자 or %"
